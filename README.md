# Analysis of Water Quality Parameters in Shrimp Culture
Data insights to support sustainable aquaculture practices

## Key Aspects of the Project
- Analysis of water quality parameters (TAN, Phosphate, Phytoplankton)
- Identification of environmental and operational drivers
- Support for sustainable shrimp farming decisions

## Methodology
- Descriptive statistical analysis
- Correlation analysis
- Regression modeling to quantify parameter relationships

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Sklearn

## Stakeholder-Centric Analysis
- Supports aquaculture operators and farm managers
- Enables early detection of water quality risks
- Improves operational decision-making

## Ethical Considerations
- Responsible interpretation of environmental data
- Avoiding misleading conclusions from incomplete records

## Deliverables
- Environmental analysis notebooks
- Water quality insights
- Operational recommendations for shrimp culture
